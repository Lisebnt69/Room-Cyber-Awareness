# Logos officiels — bibliothèque de templates phishing

Déposez ici les logos officiels des marques pour les templates d'emails.
Formats acceptés : **PNG** ou **SVG** (privilégiez PNG pour le rendu email universel).

| `urssaf.png`             | URSSAF                 | urssaf.fr                       |
| `amazon.png`             | Amazon                 | aboutamazon.com/news/company-news |

## Dimensions conseillées

- **Hauteur** : 32–48 px (le CSS redimensionne automatiquement)
- **Largeur** : proportionnelle au logo original
- **Fond** : transparent (PNG) ou couleur de la marque

## Fallback

Si un fichier est manquant, le template affiche automatiquement le nom de la marque
stylisé aux couleurs officielles (pas de rupture visuelle).

## Usage pédagogique

Ces logos sont utilisés dans un contexte de **simulation de phishing à but pédagogique**
(sensibilisation cybersécurité). Aucun email n'est envoyé à des tiers sans leur consentement
explicite dans le cadre des campagnes de formation ROOMCA.
