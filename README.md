🧠 ROOMCA

Plateforme immersive de sensibilisation à la cybersécurité — format escape game interactif

🚀 Concept

ROOMCA transforme la sensibilisation cybersécurité en expérience immersive.

Au lieu de formations chiantes ou de quiz, les utilisateurs jouent des missions type escape game :

🔍 Trouver des indices (emails, clés USB, post-it…)
📩 Analyser un phishing réaliste
🔐 Reconstituer un mot de passe
💻 Interagir avec un faux terminal (logs, commandes)
🎥 Surveiller des caméras pour détecter une intrusion

👉 Objectif : apprendre en faisant, pas en lisant.
